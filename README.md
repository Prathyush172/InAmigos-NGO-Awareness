# InAmigos Foundation Website - Multi-Page HTML/CSS Version

This updated project is a professional NGO awareness website built with only HTML5 and CSS3.

## Pages

- `index.html` - Home dashboard
- `about.html` - story, mission, vision, founder, certifications, values
- `projects.html` - Seva, Bachpanshala, Jeev, Udaan, Prakriti, and Vikas
- `gallery.html` - official image gallery with CSS hover captions
- `getinvolved.html` - volunteer, intern, partner, and support CTAs
- `contact.html` - contact details, UI-only form, social links, and map iframe

## Notes

There are no JavaScript files and no `<script>` tags. The responsive navigation uses a CSS-only checkbox menu. Open `index.html` in a browser to view the site.
