# 🌿 InAmigos Foundation - NGO Awareness Website

> A responsive NGO awareness website developed using HTML and CSS to showcase the mission, projects, and social impact of **InAmigos Foundation**.

---

## 🌐 Live Demo

🔗 https://your-vercel-link.vercel.app

---

## 📖 About

This website was created to spread awareness about the initiatives of **InAmigos Foundation**. It provides visitors with information about the NGO, its flagship projects, gallery, volunteer opportunities, and contact details in a clean and responsive interface.

---

## ✨ Features

- 🏠 Responsive Home Page
- 👨‍👩‍👧 About the Foundation
- 🌱 Six NGO Projects
- 🖼️ Photo Gallery
- 🤝 Volunteer / Get Involved Page
- 📞 Contact Page
- 📱 Fully Responsive Design
- 🎨 Modern UI with Animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Google Fonts
- Responsive Design
